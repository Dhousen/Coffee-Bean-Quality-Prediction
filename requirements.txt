# Aplikasi Prediksi Kualitas Biji Kopi Robusta

Aplikasi ini merupakan sistem prediksi kualitas biji kopi Robusta berbasis web menggunakan algoritma **Naive Bayes**. Proyek ini dikembangkan sebagai bagian dari implementasi metode data mining dalam pengambilan keputusan kualitas produk kopi.

## ðŸš€ Fitur Aplikasi

- Prediksi kualitas biji kopi Robusta (Sangat Buruk - Sangat Baik)
- Penyimpanan riwayat hasil prediksi
- Evaluasi model secara langsung dari halaman web
- Autentikasi pengguna (Login & Register)
- Visualisasi hasil prediksi dan evaluasi model
- Dilengkapi dengan teknik preprocessing dan feature engineering

---

## ðŸ§  Metodologi Data Mining

### 1. **Dataset**
- Sumber: `dataset_kopi_robusta.csv`
- Fitur utama:
  - Berat per 10 biji
  - Ukuran rata-rata per 10 biji
  - Tekstur (kasar, sedang, halus)
  - Warna (hitam, coklat tua, coklat muda, kemerahan)
  - Harga per kg
  - Kadar air (ditambahkan)
  - Label kualitas (sangat buruk - sangat baik)

### 2. **Preprocessing**
- Handling missing values (`dropna`)
- Encoding kategori: tekstur, warna, dan label
- Feature engineering: menambahkan fitur baru `berat_per_ukuran`
- Normalisasi data numerik menggunakan `StandardScaler`

### 3. **Algoritma**
- Naive Bayes Classifier (`GaussianNB`)
- Model dilatih melalui file `train_model.py`
- Model disimpan dalam `model_nb.pkl`, dan scaler di `scaler.pkl`

### 4. **Evaluasi**
- Akurasi, Precision, Recall, dan F1-Score
- Confusion Matrix
- Tersedia di halaman `/evaluasi_model`
